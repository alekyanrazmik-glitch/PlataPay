# PlataPay — дубль заявок на почту (Google Apps Script)

Сайт отправляет заявки в этот веб-скрипт (как и раньше — для Google Sheets).
Теперь скрипт вдобавок **дублирует заявку письмом** на `alekyan.razmik@gmail.com`.

Telegram сайт шлёт **сам, напрямую** — этот скрипт его не трогает и токен бота
здесь не нужен.

## Что делает

На каждую заявку:

1. пишет строку в **Google Sheets** (вкладка «Заявки», создаётся сама);
2. отправляет **копию на почту** `LEAD_EMAIL`.

## Установка (один раз)

1. Откройте ваш Apps Script (в таблице: **Расширения → Apps Script**).
2. Замените весь код содержимым [`Code.gs`](./Code.gs).
3. Если нужен другой адрес — поменяйте строку `var LEAD_EMAIL = '...'` вверху
   (можно несколько через запятую).
4. **Deploy → Manage deployments → ✎ (редактировать текущий) → Version: New
   version → Deploy.** URL `/exec` останется прежним — на сайте менять ничего
   не надо.
5. При первом запуске Google попросит разрешить доступ к таблице и почте —
   разрешите. Письма уходят с того Google-аккаунта, под которым скрипт.

## Проверка

- Откройте `/exec`-URL в браузере → `{"ok":true,"service":"platapay-mail"}`.
- Отправьте тестовую заявку с сайта → она придёт в Telegram (как обычно),
  строкой в таблицу и **письмом на почту**.

> Если письма не приходят — проверьте папку «Спам» и что при деплое выдали
> разрешение на отправку почты (scope MailApp). Бесплатный лимит Gmail —
> ~100 писем в день, для потока заявок лендинга с запасом.
